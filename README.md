# inaba-cap
INABA’S CAP has the mission 
to improve concussion
 awareness among youth athletes and
 parents in order to create 
a safer sports environment 
and ensure that athletes 
recover safely and completely. 
By sharing knowledge 
and leading community 
outreach efforts, the project
 aims to build a culture that prioritizes 
concussion safety and 
the well-being of every young athlete.
